    Reference¶

    This section contains detailed reference documentation for MaveDB's data standards, terminology, and technical infrastructure.

    In this section¶

    • Data Standards -- How MaveDB represents and exchanges data using GA4GH VRS and VA-Spec standards.
    • Score Calibrations -- How functional scores are calibrated to clinical evidence strength using baseline scores, functional classifications, and evidence strengths.
    • Accession Numbers -- How MaveDB URNs are structured for experiment sets, experiments, score sets, and other resources.
    • Controlled Vocabulary -- Definitions of the standardized terms used in MaveDB metadata and assay facts.
    • Permissions -- Who can view, edit, publish, and delete each type of resource in MaveDB.
    • Variant Mapping -- How MAVE variants are mapped from assay-specific target sequences to human reference genome coordinates.
    • Assay Facts -- Structured summaries of experimental assay properties displayed on score set pages.
    • Collections -- How datasets are organized into curated collections in MaveDB.
